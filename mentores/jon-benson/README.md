## As grandes ideias do Jon (resumo)

1. **80% do esforço nos 2 primeiros blocos** da VSL (atenção + conexão emocional).
2. **Um avatar, uma pessoa, um CTA.** Sempre comunicação 1:1.
3. **Compelir, não coagir.** Vender é ético quando o produto ajuda de verdade.
4. **Emoção compra, lógica justifica** — desejos (*wants*) no primeiro minuto; lógica/preço/FAQ no fim.
5. **"Não existe copy longa, existe copy chata."**
