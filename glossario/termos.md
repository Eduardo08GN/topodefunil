# Glossário — Direct Response

Termos e jargão. Cresce conforme aparecem no material.

| Termo (EN) | Em PT | O que é |
|------------|-------|---------|
| Big Idea | Grande ideia | O conceito central e novo que sustenta a peça inteira |
| Lead | Abertura | Os primeiros parágrafos/minutos que fisgam e prometem |
| Unique Mechanism | Mecanismo único | O "porquê" novo do problema e da solução |
| VSL | Carta de vendas em vídeo | *Video Sales Letter* |
| Hook | Gancho | O que prende a atenção na primeira frase/cena |
| Swipe | Swipe / referência | Trecho de copy real guardado pra estudar e reaplicar |
| Offer | Oferta | A soma de produto + bônus + preço + garantia + escassez |
| RMBC | — | Método de Stefan Georgi: Research, Mechanism, Brief, Copy |
| CSL | Carta de vendas "conteúdo" | Anúncio disfarçado de conteúdo nativo (notícia, podcast, vídeo orgânico) pra baixar a guarda |
| Advertorial | Publieditorial | Anúncio em formato de artigo/matéria |
| Proof element | Elemento-prova | Prova concreta que sustenta a promessa (Bencivenga: a maior alavanca) |
| Fascination | Fascínio | Bullet de curiosidade que promete uma revelação sem entregá-la |
| AOV | Ticket médio | *Average Order Value* — valor médio por pedido |
| Reason why | Porquê | A razão lógica/explicação que torna a promessa crível |
| Future pacing | Projeção futura | Fazer o prospecto se imaginar já vivendo o resultado |
| Swipe file | Biblioteca de referências | Coleção de copy real guardada pra estudar/reaplicar |
| Solo offer | Oferta única | Uma só oferta focada (Kennedy) em vez de catálogo, pra aquisição |
| Hook | Gancho (vídeo) | Os primeiros 3–5s do reel/vídeo que decidem tudo |
| Open loop / Curiosity gap | Laço aberto / lacuna de curiosidade | Informação incompleta que a mente quer fechar |
| Pattern interrupt | Interrupção de padrão | Quebra da expectativa que trava o scroll |
| Key visual | Visual-herói | A imagem-âncora dos primeiros segundos (escolhida antes da fala) |
| Retention graph | Curva de retenção | % de gente ainda assistindo a cada segundo — sinal-chave pro algoritmo |
| Match cut | Corte casado | Emenda em que o fim de uma cena "casa" com o início da próxima |
| Viral sense | Faro de viral | Intuição treinada (como um dataset) do que viraliza |
| Drunk grandma | Vó de 90 anos | Regra: entendível em meio segundo, sem contexto extra |
| Rubbernecking | "Pescoço de borracha" | Impulso de olhar pra tragédia/drama |

*(adicionar conforme surgir)*
